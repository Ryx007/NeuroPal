What changed / mapping for the dev

  • D1 — Drawer nav: AppTabBar removed; TAB_CONFIG becomes the drawer list (adds Toolbox D7 + Settings D2). Opens via hamburger (was: opened Tweaks) or edge-swipe-right; closes on scrim tap / swipe-left / destination select. Every destination stays navigation.navigate(routeName).
  • D3 — Liquid glass: .glass/.glass--deep = real blur + ruby tint + specular top edge = the extended GlassPanel (was 0.82 flat tint). Applies to header, drawer, toast (and player/sheets on other screens).
  • D4 — Safe area + status bar: content wrapped in safe insets; status strip filled with a solid theme tint (never transparent over content); light icons on dark.
  • D5 — Toasts: ✕ dismiss + swipe-to-dismiss + 5s auto-hide (progress line). Maps to every Toast.show(...); success toasts may auto-hide faster.
  • Preserved: ● NeuroPal wordmark (live DataPulse dot), profile avatar → Profile, all accessibility labels. No new color roles introduced.
